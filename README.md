# Thyrocare Nagercoil

Blood test booking platform for Thyrocare Nagercoil — an authorized Thyrocare collection centre serving Nagercoil and Kanyakumari district.

## Tech Stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Lovable Cloud (backend)

## Development

```sh
npm i
npm run dev
```
